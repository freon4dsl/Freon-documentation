# Projection Fragments

Example:

```
MyConcept {[
    some text in front of fragment XX [fragment xx].
    some text in front of fragment YY [fragment yy].
]
fragment xx [
    This is fragment XX.
    [fragment zz]
]
fragment yy [
    This is fragment YY.
]
fragment zz [
    This is fragment ZZ.
]
}
```
